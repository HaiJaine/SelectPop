SelectPop Portable
==================

1. Extract the ZIP to any writable folder.
2. Run SelectPop.exe directly.
3. Runtime data is created next to the executable in the data/ folder.
4. Keep SelectPop.exe and the data/ folder together when moving or backing up the app.
5. For source build, setup, WebDAV sync, startup, and troubleshooting details, see the repository README.md.
